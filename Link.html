Link:

  1. Absolute Link:
    It allows to visit from one website to another.
    Example:
    Google
    Yahoo
  2. Relative Link:
    It allows to visit multiple pages within same website.
    Example:
    Home Page
    List Page
    Nested page
    Netlify Page
  3. On Page Link:
    4. It allows to visit from one part of the page to another part/section.
